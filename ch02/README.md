# Ch01. 객체지향 프로그래밍

1. 영화 예매 시스템
2. 객체지향 프로그래밍을 향해
    - 자율적인 객체
        - 중요한 사실 두 가지
            - 객체는 상태(state)와 행동(behavior)을 함께 가지는 복합적인 존재
            - 객체는 스스로 판단하고 행동하는 자율적인 존재
        - 캡슐화/접근 제어는 객체를 두 부분으로 나눔
            - 퍼블릭 인터페이스 (Public Interface): 외부에서 접근 가능한 부분
            - 구현 (Implementation): 내부에서만 접근 가능한 부분

4. 상속과 다형성
